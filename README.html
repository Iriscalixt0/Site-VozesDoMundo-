🎶 Vozes do Mundo - Um Projeto Fictício de Música e Educação 🎤

✨ O Que é o Vozes do Mundo?

Bem-vindo ao Vozes do Mundo, um site fictício criado com muito carinho para fins de estudo no desenvolvimento web! 🌍🎶 Este projeto foi feito para explorar conceitos básicos de HTML e CSS enquanto cria uma "escola de música" fictícia. O objetivo é proporcionar uma experiência de navegação simples, funcional e visualmente interessante, sem a complexidade de várias páginas – focando exclusivamente na página inicial.

Aqui, você vai encontrar informações sobre a escola, um formulário de inscrição e um menu de navegação interativo (mas sem links, já que a ideia é focar na página inicial mesmo). Vamos mostrar como o design pode ser limpo e responsivo, funcionando bem em qualquer dispositivo. E, claro, o projeto é todo fictício – só para aprendizado! 😉

🚀 O Que Você Vai Encontrar Aqui?

  • Formulário de Inscrição: Um formulário simples para que os "interessados" se inscrevam. Preencha nome, e-mail, telefone e estado (mas lembre-se, isso é só para estudo, então nada será enviado de fato 😉).
  • Menu Interativo (sem links): Um menu de navegação que é visualmente interativo, mas não leva a páginas diferentes, já que estamos focados em uma única página: a página inicial.
  • Conteúdo Informativo: Algumas seções mostrando informações fictícias sobre a escola de música, como a missão, a metodologia e o impacto dos cursos. É tudo sobre mostrar como podemos estruturar um site simples com conteúdo claro e direto.
  • Design Responsivo: O site foi criado para se adaptar a diferentes dispositivos. Teste em celulares, tablets ou desktops – a página se ajusta bem a qualquer tela.

💻 Como Usar

Para testar o projeto localmente ou brincar com ele, basta clonar o repositório:

git clone https://github.com/Iriscalixt0/Site-VozesDoMundo.git

Depois, abra o arquivo index.html no seu navegador para ver o resultado!

🛠️ Tecnologias Usadas

Aqui estão as tecnologias que usei para criar esse projeto simples, mas visualmente atrativo:

  • HTML5: Estruturei o site com HTML para definir as seções e os campos do formulário.
  • CSS3: O design e os estilos foram feitos com CSS, incluindo responsividade e interatividade no menu.
  • Fontes Web: A fonte Inter foi utilizada para garantir uma leitura fácil e moderna.
  • Responsividade: O layout foi pensado para funcionar bem em qualquer dispositivo, usando media queries para ajustar a exibição em diferentes tamanhos de tela.
